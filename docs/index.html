TzText

The world's timezones in plain text, JSON, JavaScript arrays, and HTML select lists.

About

This project was started because I needed a way to select zoneinfo paths from a web site. I was using static html (no PHP) and JavaScript only. I quickly found that JavaScript has no concept of timezones other than what it detects from the browser. I did not want to use a heavyweight library like JQuery, so I ended up creating my own.

What is tzText?

  • A JavaScript library called tzNamesLib.js
  • HTML select lists of areas and locations.
  • JSON arrays of areas and locations.
  • Plain text lists of areas and locations.
  • A shell script that extracts names from the timezone database.

What tzText is not

  • tzText has no concept of timezone rules or daylight saving changes. It is only a list of areas and locations.
